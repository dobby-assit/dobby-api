# api
[![Build Status](https://travis-ci.org/optly/api.svg?branch=master)](https://travis-ci.org/optly/api)

Optly web api

## Getting Started

### Build

Once you have cloned the repo and have installed
leiningen.

Run:

```
  lein ring server
```

In order to launch locally.


### Testing

We use leiningen for testing.

Run:

```
  lein test
```
